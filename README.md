## 💻 Projets

### Reminder Famille
> Application web full-stack de gestion des tâches familiales.
> React 18 + Vite (front) / Node.js + Express + mysql2 (back) / MariaDB.

🔗 **Code source** : https://github.com/Vladimir08880888/reminder-famille

Fonctionnalités clés :
- Familles avec rôles parent / enfant + validation parent
- Tâches récurrentes (daily / weekly / monthly / yearly)
- Export iCal natif (notifications iPhone & Android sans application)
- Statistiques, calendrier mensuel, commentaires sur tâches
- 57 tests d'intégration automatisés
